## 9. FAQ (질문 받으면 복붙)

- **무료인가요?** 네, 전 기능 무료, 광고 없음. 개인 프로젝트입니다.
- **기록 어디 저장돼요?** 내 기기(브라우저). 설정→내보내기로 백업/복원, CSV(엑셀)도 지원.
- **앱스토어엔 없어요?** 웹앱입니다. 브라우저 "홈 화면에 추가"로 앱처럼 사용. 반응 좋으면 스토어 검토.
- **아이폰/PC 돼요?** 브라우저만 있으면 전부 됩니다.
- **모임은 어떻게?** 모임 탭에서 방 만들기 → 코드 공유 → 닉네임만으로 참여.
